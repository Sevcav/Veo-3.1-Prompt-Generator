About position locking
When characters are added in the ElevenLabs tab, the prompt builder injects a CHARACTER POSITIONS block anchoring every action beat and every silent beat to screen position (LEFT / RIGHT / CENTER) rather than character names. This prevents Veo from swapping which character owns each action — the most common failure mode in two-character scenes.
