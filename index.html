About position locking
When multiple characters are added in the ElevenLabs tab, the prompt builder injects a CHARACTER POSITIONS block that anchors every action beat and every silent beat to screen position (LEFT / RIGHT / CENTER) rather than character names. This prevents Veo from swapping which character owns each action — a common failure mode with two-character scenes.
